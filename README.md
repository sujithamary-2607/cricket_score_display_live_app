# 🏏 Cricket Live Score Display App

## 📌 Project Overview

The Cricket Live Score Display App is a Python-based application that provides **real-time cricket information** including live scores, team squads, and match schedules across multiple formats such as IPL, T20 World Cup, and ODI.

This project allows users to easily access up-to-date cricket data through a simple and interactive interface.

---

## 🚀 Features

### 📊 Live Score Updates

* View **real-time cricket scores**
* Covers multiple formats:

  * IPL
  * T20 World Cup
  * ODI matches

---

### 📅 Match Schedule

* Displays upcoming and ongoing match schedules
* Organized by tournament and format

---

### 👥 Team Squad Details

* View **team player lists (squads)**
* Helps users understand team composition

---

### 🌍 Multi-Format Support

* Supports:

  * T20
  * ODI
  * IPL tournaments

---

### ⚡ Simple Execution

* Run the application using a single command:

```
python app.py
```
